# /save-session — 세션 요약 저장

컨텍스트 70% 도달 시 반드시 실행.
다음 세션이 이 파일 하나만 읽으면 맥락을 이어받을 수 있게 작성.

저장 위치: `.claude/sessions/YYYY-MM-DD-[주제].md`

## 저장 형식
```markdown
# 세션 — YYYY-MM-DD [주제]

## 완료
- [x] 구체적인 작업 내용

## 미완료 / 다음 세션
- [ ] 남은 작업

## 발견한 패턴·주의사항
- 이 세션에서 겪은 문제와 해결 방법

## 파일 상태
- taskflow.html: 약 X,XXX줄
- colspan: 11
- 마지막 수정 함수: ...
```

## 언제 실행하나
- 컨텍스트 70% 이상 차면 반드시
- 큰 기능 완료 후
- 세션 종료 전
