# /add [기능 설명]

기능 추가 워크플로우. **승인 없이 진행 금지.**

## 실행 순서

### Step 1 — 승인 게이트 (필수)
```
[PLAN]
추가 기능: ...
삽입 위치: [함수명 / 줄 범위]
변경 범위: CSS / HTML / JS
다른 함수 영향: [있으면 명시]
승인하시면 진행합니다.
```

### Step 2 — 승인 후 진행
```
1. grep으로 삽입 위치 확인
2. 최소 범위만 수정
3. /verify
4. 결과 보고
```

## 예시
```
/add 업무 간 연결 기능
/add 대시보드 UI 개편
/add 캘린더 주간 뷰
```
