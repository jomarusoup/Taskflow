- colspan 현재 **11** — 컬럼 추가 시 모든 colspan 동시 수정
- `ep-tags-${t.id}` ID 중복 발생 이력 — 수정 시 반드시 grep 확인
- `field-row`는 1fr 1fr 2컬럼 — 3개 넣으면 레이아웃 깨짐
- `grpSort{}` 객체: 카테고리별 독립 정렬 상태 관리
- `contactSort{}` 객체: 연락처 정렬 상태
