≡ 업무 대장
▶ 인라인 편집제목 옆 ▶ 클릭으로 시작일·마감일·태그·카테고리·우선순위·상태·담당자를 행 아래에서 바로 편집합니다.
그룹 뷰카테고리별로 업무를 그룹화하여 표시합니다. 각 그룹 헤더 클릭으로 접기/펼치기가 가능합니다.
복합 필터 / CSV다양한 조건으로 업무를 검색하고, `↓ CSV` 버튼으로 필터링된 목록을 엑셀용 CSV로 내보냅니다.
📦 인벤토리
대장 관리좌측 사이드바에서 여러 개의 대장을 생성하고, `⠿` 손잡이를 드래그하여 순서를 변경합니다.
탭 시스템대장 내부에 여러 탭을 만들어 데이터를 분류하며, 탭 제목의 `⠿` 손잡이로 순서를 바꿀 수 있습니다.
멀티 탭 엑셀`↓ 내보내기 (Excel)` 클릭 시 내보낼 대장과 탭을 선택할 수 있으며, 각 탭은 엑셀의 개별 시트(Sheet)로 생성됩니다.
👤 연락처
카테고리별 정/부한 연락처가 여러 카테고리에 속할 수 있으며, 각 카테고리마다 정/부 담당 여부를 지정합니다.
엑셀 내보내기`↓ Excel` 버튼으로 연락처 목록을 내보냅니다. 관리 편의를 위해 수정·삭제 버튼은 항상 표시됩니다.
⚙ 설정 및 데이터
커스텀 설정카테고리, 태그, 우선순위, 상태의 이름과 색상을 자유롭게 변경하여 시스템에 즉시 반영합니다.
JSON 백업`🔒 지금 백업` 버튼으로 전체 데이터(인벤토리 포함)를 JSON 파일로 저장하고, 백업 탭에서 복원할 수 있습니다.
